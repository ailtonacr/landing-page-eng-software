Comunidade de Engenharia de Software para quem vive tecnologia

Engenharia de Software é uma comunidade brasileira no WhatsApp para estudantes, desenvolvedores iniciantes e entusiastas trocarem conhecimento sobre programação, desenvolvimento de software, carreira tech, IA, DevOps, arquitetura de software e mercado de tecnologia.

O grupo de Engenharia de Software é gratuito e aberto também a estudantes de Ciência da Computação, Sistemas de Informação, Análise e Desenvolvimento de Sistemas e outras áreas de tecnologia.

Grupo de programação e comunidade de tecnologia

Participe para tirar dúvidas, conversar sobre backend, frontend, full stack, cloud e infraestrutura, compartilhar projetos, portfólio e GitHub, encontrar materiais e fazer networking.
